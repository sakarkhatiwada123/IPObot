	Steps to Follow:

1. Install Python on your system.
2. Install google chrome in your system. If you already have, update it.
3. Open terminal / command prompt and run the following commands.

	pip install selenium
	pip install webdriver-manager

4. Open credentials.txt file and update it with necessary inputs and credentials (DPID, username,         password, CRN number and PIN). Separate all the inputs with semi-colomn ( ; ). 
5. Open terminal / command prompt inside the IPObot folder.
6. Run the below command
	python main.py
